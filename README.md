### NexusHR: Employee Resource and Skill Management Portal 
NexusHR is a centralized web portal designed for real-time skill tracking and resource management, helping companies reduce bench time and optimize talent allocation.

### Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/en) installed on your system.

### Installation

1.  Clone this repository to your local machine.
2.  Open your terminal in the project root directory and run the following command to install the required dependencies:

bash

    npm install

### Running the Development Server

To start the local development server, run:

bash

    npm run dev
    

Use code with caution.

###  Problem

Companies often struggle to track employee skills in real-time, leading to increased bench time, sub-optimal team utilization, and delays in finding the best-fit talent for upcoming projects.

### Solution

**NexusHR** solves this by providing a unified platform for centralized skill tracking and intelligent resource management, allowing organizations to deploy the right talent at the right time.

* * *

### ✨ Key Features

*   **📊 Overview & Admin Dashboard:** Live metrics displaying total, available, and assigned resources.
*   **🤖 AI Matchmaker:** Automatically suggests the best-fit employees for projects based on skill-match percentages.
*   **📉 Bench Analytics:** Real-time visibility into available, utilized, and benched employees.
*   **👥 Employee Directory:** Centralized, searchable repository of all personnel and their skill profiles.
*   **🛡️ Security Audit:** Comprehensive tracking and compliance monitoring for safe resource data handling.
*   **🔔 Notifications System:** Live updates for project matches, bench alerts, and resource reallocations.
